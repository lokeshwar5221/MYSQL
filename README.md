# MYSQL
🚀 Welcome to my **SQL Learning Journey**! This repository contains my daily SQL learning notes, practical examples, and hands-on exercises covering DBMS, RDBMS, SQL commands, constraints, joins, functions, transactions, and more. A step-by-step journey to strengthen my database and backend development skills.

#DAY1
This SQL learning journey begins with the fundamentals of databases and Full Stack Development. It covers key concepts in Frontend, Backend, APIs, Storage Systems, File Management System (FMS), Database Management System (DBMS), and SQL. The focus is on building a strong foundation in database management through consistent learning and hands-on practice.

#DAY2
The basics of DBMS, including FMS vs DBMS comparison, database fundamentals (Data, Field, Record, Database), DBMS features, Client-Server Architecture, and different types of Data Models such as Physical, Logical, Hierarchical, Network, and Relational Data Models (RDBMS). Also explored relational database concepts like tables, rows, columns, and relationships between tables.

#DAY3
SQL basics including SQL Identifiers, rules for naming database objects, and introduction to SQL and its commands. Covered different categories of SQL commands such as DDL, DML, DCL, TCL, and DQL along with their commands and uses. Also learned database and table creation, deletion, and basic SQL syntax for creating and managing database objects.

#DAY4
practical SQL operations using MySQL, including database creation, deletion, selection, and viewing database objects. Practiced creating tables with different data types and explored table structure using the DESC command. Covered ALTER TABLE operations such as adding new columns, modifying column data types, changing column names, dropping columns, and renaming tables. Also worked with different MySQL data types like CHAR, VARCHAR, INT, DATE, DECIMAL, ENUM, SET, and TINYTEXT while designing database tables.
