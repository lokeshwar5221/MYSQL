🚀 20 Days SQL Learning Journey

Welcome to my 20 Days SQL Learning Journey! 🎯

This repository documents my daily progress as I learn SQL and Database Management Systems (DBMS) from the ground up. Each day focuses on building a strong understanding of database concepts through theory, practical examples, and hands-on MySQL exercises. My goal is to strengthen my SQL skills and establish a solid foundation for Full Stack Development.

📅 Day-wise Learning Summary
🌟 Day 1 – Introduction to SQL & Database Fundamentals

Started my SQL journey by exploring the fundamentals of Full Stack Development and understanding where databases fit into modern applications. Learned the basics of Frontend, Backend, APIs, Storage Systems, and the evolution from File Management Systems (FMS) to Database Management Systems (DBMS). This session laid the groundwork for understanding how SQL powers efficient data management.

🌟 Day 2 – DBMS Fundamentals & Data Models

Built a strong understanding of Database Management Systems (DBMS) by comparing FMS vs DBMS and learning essential database concepts such as Data, Field, Record, and Database. Explored Client-Server Architecture, DBMS features, and various Data Models including Physical, Logical, Hierarchical, Network, and Relational (RDBMS). Also gained insight into relational database concepts like tables, rows, columns, and relationships.

🌟 Day 3 – SQL Fundamentals & SQL Commands

Learned the core concepts of Structured Query Language (SQL), including SQL Identifiers and the rules for naming database objects. Explored the five major categories of SQL commands—DDL, DML, DCL, TCL, and DQL—along with their purposes and commonly used statements. Practiced creating and deleting databases and tables while understanding the syntax used to define and manage database objects.

🌟 Day 4 – MySQL Hands-on: Table Management & ALTER Operations

Focused on practical implementation using MySQL by creating, deleting, and selecting databases and tables. Learned to inspect table structures using the DESC command and performed various ALTER TABLE operations such as adding, modifying, renaming, and dropping columns, as well as renaming tables. Also gained hands-on experience with commonly used MySQL data types including CHAR, VARCHAR, INT, DATE, DECIMAL, ENUM, SET, and TINYTEXT, strengthening my understanding of database schema design.
